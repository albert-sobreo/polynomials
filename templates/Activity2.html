  1. ) x\(^2\) + 2x = 3

    2.   steps:
        A.   Add the square of one-half of the
              coefficient of x on both sides of the
              equation.
              a = \({2\over2}\)
              a = 1
              \(a^2 = 1^2\)
              \(a^2 = 1\)
        B.   therefore:
              \(x^2 + 2x + 1 = 3 + 1\)
              \(x^2 + 2x + 1 = 4\)
              (x + 1)\(^2\) = 4
              √(x + 1)\(^2\) = √4
        C.   first solution:
              x + 1 = 2
              x = 2 - 1
              x = 1
        D.   second solution:
              x + 1 = -2
              x = -2 - 1
              x = -3

  2. ) x\(^2\) - 4x = 12

    3.   steps:
        A.   Add the square of one-half of the
              coefficient of x on both sides of the
              equation.
              a = -\({4\over2}\)
              a = -2
              \(a^2 = (-2)^2\)
              \(a^2 = 4\)
        B.   therefore:
              \(x^2 - 4x + 4 = 12 + 4\)
              (x - 2)\(^2\) = 16
              √(x - 2)\(^2\) = √16
              x - 2 = ± 4
        C.   first solution:
              x - 2 = 4
              x = 4 + 2
              x = 6
        D.   second solution:
              x - 2 = -4
              x = -4 + 2
              x = -2

  3. ) a\(^2\) - 3a - 40 = 0

    4.   steps:
        A.   Add the square of one-half of the
              coefficient of x on both sides of the
              equation.
              x = -\({3\over2}\)
              x\(^2\) = - \({3\over2}^2\)
              x\(^2\) = \({9\over4}\)
        B.   therefore:
              a\(^2\) - 3a + \({9\over4}\) = 40 + \({9\over4}\)
              √(a - \({3\over2}\))\(^2\) = √\({169\over4}\)
              a - \({3\over2}\) = ± \({13\over2}\)
        C.   first solution:
              a - \({3\over2}\) = \({13\over2}\)
              a = \({3\over2}\) + \({13\over2}\)
              a = \({16\over2}\)
              a = 8
        D.   second solution:
              a - \({3\over2}\) = -\({3\over2}\)
              a = - \({13\over2}\) + \({3\over2}\)
              a = - \({10\over2}\)
              a = -5

  4. ) 2x\(^2\) + 5x - 3 = 0

    5.   steps:
        A.   Divide both sides with the
              coefficient of x\(^2\)
              \({2x^2 + 5x\over 2}\) = \({3\over2}\)
              x\(^2\) + \({5\over2}\) x = \({3\over2}\)
        B.
              a = \({5\over4}\)
              a\(^2\) = \({25\over16}\)
        C.   x\(^2\) + \({5\over2}\) x + \({25\over16}\) = \({3\over2}\) + \({25\over16}\)
              (x + \({5\over4}\))\(^2\) = \({49\over16}\)
              √(x + \({5\over4}\))\(^2\) = √\({49\over16}\)
              x + \({5\over4}\) = ± \({7\over4}\)
             
        D.   first solution:
              x + \({5\over4}\) = \({7\over4}\)
              x = \({7\over4}\) - \({5\over4}\)
              x = \({2\over4}\)
              x = \({1\over2}\)
        E.   second solution:
              x + \({5\over4}\) = - \({7\over4}\)
              x = - \({7\over4}\) - \({5\over4}\)
              x = - \({12\over4}\)
              x = -3

  5. ) x\(^2\) + 4x - 45 = 0

    6.   steps:
        A.
              a = 2
              a\(^2\) = 4
        B.
              x\(^2\) + 4x + 4 = 49
              √(x+2)\(^2\) = √49
              x + 2 = ± 7
        C.   first solution:
              x + 2 = 7
              x = 7 - 2
              x = 5
        D.   second solution:
              x - 2 = 7
              x = 7 + 2
              x = 9
