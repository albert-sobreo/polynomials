  1. ) y\(^2\) + 4y = 96

    2.   steps:
        A.
              a = 2
              a\(^2\) = 4
        B.
              y\(^2\) + 4y + 4 = 100
              √(y + 2)\(^2\) = √100
              y + 2 = ± 10
        C.   first solution:
              y + 2 = 10
              y = 10 - 2
              y = 8
        D.   second solution:
              y + 2 = -10
              y = -10 - 2
              y = -12

  2. ) x\(^2\) - 8x = 0

    3.   steps:
        A.
              a = - 4
              a\(^2\) = 16
        B.
              x\(^2\) - 8x + 16 = 16
              √(x-4)\(^2\) = √16
              x - 4 = ± 4
        C.   first solution:
              x - 4 = 4
              x = 4 + 4
              x = 8
        D.   second solution:
              x - 4 = - 4
              x = - 4 + 4
              x = 0

  3. ) 5x\(^2\) + 3x - 2 = 0

    4.   steps:
        A.
              (5x\(^2\) + 3x) / 5 = 2/5
              x\(^2\) + (3x/5) = 2/5
        B.
              a = 3/10
              a\(^2\) = (9/100)
             
        C.
              x\(^2\) + (3x/5) + (9/100) = (2/5) + (9/100)
              x\(^2\) + (3x/5) + (9/100) = (40/100) + (9/100)
              √(x + (3/10))\(^2\) = √(49/100)
              x + (3/10) = ± (7/10)
        D.   first solution:
              x + (3/10) = (7/10)
              x = (7/10) - (3/10)
              x = 4/10
              x = 2/5
        E.   second solution:
              x + (3/10) = - (7/10)
              x = - (7/10) - (3/10)
              x = - (10/10)
              x = - 1
