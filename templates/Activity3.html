  1. ) y\(^2\) + 4y = 96

    2.   steps:
        A.
              a = 2
              a\(^2\) = 4
        B.
              y\(^2\) + 4y + 4 = 100
              √(y + 2)\(^2\) = √100
              y + 2 = ± 10
        C.   first solution:
              y + 2 = 10
              y = 10 - 2
              y = 8
        D.   second solution:
              y + 2 = -10
              y = -10 - 2
              y = -12

  2. ) x\(^2\) - 8x = 0

    3.   steps:
        A.
              a = - 4
              a\(^2\) = 16
        B.
              x\(^2\) - 8x + 16 = 16
              √(x-4)\(^2\) = √16
              x - 4 = ± 4
        C.   first solution:
              x - 4 = 4
              x = 4 + 4
              x = 8
        D.   second solution:
              x - 4 = - 4
              x = - 4 + 4
              x = 0

  3. ) 5x\(^2\) + 3x - 2 = 0

    4.   steps:
        A.
              \({5x^2 + 3x\over5} = {2\over5}\)
              \(x^2 + {3x\over5} = {2\over5}\)
        B.
              a = \({3\over10}\)
              \(a^2 = {9\over100}\)
             
        C.
              \(x^2 + {3x\over5} + {9\over100} = {2\over5} + {9\over100}\)
              \(x^2 + {3x\over5} + {9\over100} = {40\over100} + {9\over100}\)
              √\((x + {3\over10})^2\) = √\({49\over100}\)
              \(x + {3\over10}\) = ± \({7\over10}\)
        D.   first solution:
              \(x + {3\over10} = {7\over10}\)
              \(x = {7\over10} - {3\over10}\)
              x = \({4\over10}\)
              x = \({2\over5}\)
        E.   second solution:
              \(x + {3\over10} = - {7\over10}\)
              \(x = - {7\over10} - {3\over10}\)
              \(x = - {10\over10}\)
              x = - 1
